# 餐廳市場分析系統 (Store Strategist)

一個用於餐廳市場分析和競爭情報收集的Java應用程式。

## 功能

- 收集並分析餐廳評論數據
- 比較競爭對手的表現
- 視覺化評分和評論資訊
- AI輔助分析建議
- 自訂主題和界面設定

## 技術架構

- Java + JavaFX
- Google Places API
- JSON數據處理
- Python數據收集工具

## 系統需求

- Java 11或更高版本
- Gradle建置工具
- 網路連接（用於API功能）

## 安裝與使用

1. 複製專案
```
git clone https://github.com/ntoujavaproject/store-strategist.git
```

2. 建置專案
```
./gradlew build
```

3. 執行應用程式
```
./gradlew run
```
